# [D-01] 周回ループ実装

## 背景
Residueのコア体験は「1周完結 -> 継承」で成立する。

## 目的
世界選択から周回終了までを1ループとして成立させる。

## スコープ
- 世界選択
- 探索（最小イベント遷移）
- 周回終了判定（仮: End Runボタン）
- 継承反映（仮: 魂ポイント加算）

## 実装タスク
1. Run中の状態遷移（探索中/終了）を実装
2. 周回終了時に結果を集計
3. `MetaState` に継承値を反映
4. 次周開始時に反映値を表示

## 完了条件
- 1周終了で魂ポイントが増える
- 次周開始時に増加が確認できる

## 検証手順
1. 1周プレイして終了
2. ポイント増加を確認
3. もう1周して累積を確認
