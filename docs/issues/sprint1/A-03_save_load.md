# [A-03] セーブ/ロード基盤

## 背景
周回とフラグ管理が中核のため、永続化は初期に固定する必要がある。

## 目的
`MetaState`, `RunState`, `Flags` を保存・復元できるようにする。

## スコープ
- セーブファイル定義
- 保存API（手動保存、周回終了時保存）
- 読み込みAPI（起動時復元）

## 実装タスク
1. `SaveService` 実装
2. 状態モデルのシリアライズ/デシリアライズ
3. 起動時ロード + 破損時フォールバック
4. 世代管理（最新 + 1世代前）

## 完了条件
- 再起動して状態が復元される
- 破損ファイル時に1世代前から復旧できる

## 検証手順
1. 状態を変更して保存
2. 再起動で同一状態を確認
3. 最新セーブを壊して起動し、前世代復旧を確認
