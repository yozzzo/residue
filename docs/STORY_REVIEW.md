# Residue ストーリーレビュー

**レビュー日**: 2026-02-17
**対象**: events.json, cross_links.json, enemies.json, worlds, jobs, locales

---

## 1. 総評

Residueの物語基盤は「死の痕跡が残る」というコアコンセプトをしっかり体現しており、特に周回数に応じたreaction_slotの段階開示設計は優れている。しかし、**現状のイベント群は「発見→情報提示→ボス」の一本道が強く、ストーリーの連続性・選択肢の意味・感情的フックに改善余地がある**。

---

## 2. 問題点リスト

### 2.1 ストーリーの連続性：一話完結しすぎ

| イベント | 問題 |
|---------|------|
| `m_e03` (礼拝堂) | 紋章を調べても何も起きない。sets_flagがなく、後続イベントに繋がらない |
| `m_e06` (地下墓地入口) | 手形を調べてもdiscoveryはあるがフラグなし。死者の痕跡という核テーマに直結する場所なのに使い捨て |
| `f_e03` (整備区画) | N-06の日記は良い伏線だが、N-06についてそれ以上掘り下がらない。日記を読んだフラグの活用がf_e04のreaction_slotのみ |
| `f_e05` (研究棟) | 培養槽を調べても「お前はN-07」という情報で完結。N-08以降の空の槽が後の展開に繋がらない |

**根本的な問題**: 多くのイベントが「情報を提示して終わり」。プレイヤーが「次にこの情報をどう使えるか」と思える引きがない。

### 2.2 選択肢の意味：フレーバーに留まる選択が多い

| イベント | 選択肢 | 問題 |
|---------|--------|------|
| `m_e04` (廃屋の影) | 「戦う」「逃げる」 | 戦闘か逃走の二択のみ。mercifulタグのreaction_slotがあるのに「話しかける」選択肢がない |
| `m_e07` (封印の間) | 「封印を強化する」 | 強化しても何も起きない（フラグなし）。obedientな選択に報酬がない |
| `f_e01` (廃棄セクター入口) | 「侵入する」「ログを確認する」 | 構造はm_e01と同じ。2つの世界の入口が同じパターンで新鮮さがない |
| `f_e06` (シェルター) | speaker: "warrior" になっているが、この世界にwarriorは不自然 |

### 2.3 伏線と回収：張りっぱなしの伏線

| 伏線 | 状態 |
|------|------|
| 村長の「何度目だ？」 | 周回メタ知識の暗示だが、村長との関係がm_e02/m_e02bで終わる。村長自身が何者なのかが掘り下げられない |
| 井戸の声 (m_e05) | well_peekedフラグはm_e06で1行使われるだけ。「自分の声」という設定の回収がない |
| N-06の存在 | 日記（f_e03）と培養槽（f_e05）で言及されるが、N-06が何をしたか・なぜ失敗したかの物語がない |
| 封印の間の二つの声 (m_e07) | truth_stage 2で「司教の——」と匂わせるが、回収イベントがない |
| 鎧の記憶 (f_foreign_knight_03) | 「この場所を覚えている」の後が続かない |

### 2.4 世界観の深み：クロスリンクが機械的

- 現在のクロスリンクは2本あるが、**どちらも「アイテムを持っていく→渡す→真実が語られる」という同じパターン**。
- 中世の司教と未来のコア預言者の関係性が示唆されるが、直接的な因果（同一存在？対立関係？）が曖昧。
- **ジョブ持ち込み時のイベント（foreign系）がフレーバー止まり**。騎士で未来に行っても、サイボーグで中世に行っても、「珍しがられる」だけでストーリーが変わらない。

### 2.5 感情的なフック：NPCへの感情移入が弱い

- **名前のあるNPCがいない**。村長は「老人」、修道士は「霊」、敵も一般名詞。
- プレイヤーが守りたい・救いたいと思える存在がいない。
- 「何度も死ぬ」設定だが、死ぬことへの感情的コストが低い（失うものが数値だけ）。
- **N-06は感情フックになりうる最大の素材**だが、現状では「前のモデル」という情報でしかない。

## 3. 改善案

### 3.1 連鎖イベントの導入：「未完の物語」システム

**方針**: 一部のイベントを2〜3段階の連鎖にし、同一周回内または次周回で続きが発生するようにする。

#### 案A: 村長の真実（中世・3段階）
- **段階1** `m_e02` (既存を拡張): 村長が「何度目だ？」と聞く → 地下の話で終わらず、「娘のことを聞いてくれるか」という新選択肢を追加
- **段階2** `m_e02c` (新規): 村長の娘が井戸に身を投げた話。娘の名前は「エリス」。実はエリスはResidueプロジェクトの最初の被験体
- **段階3** `m_e02d` (新規): 周回7以降、村長自身がResidueであることが判明。「わしも…何度目かわからん」

#### 案B: N-06の遺言（未来・3段階）
- **段階1** `f_e03` (既存を拡張): 日記の最後に暗号化されたメッセージが追加。解読にはterminal_hackedフラグが必要
- **段階2** `f_e03b` (新規): 解読すると、N-06がコア預言者を倒す方法を発見していたことがわかる。だが「代償が大きすぎる」と書いて止めた
- **段階3** `f_e03c` (新規): N-06の最後の記録。「N-07へ。お前がこれを読んでいるなら、俺は失敗した。だが、俺の失敗もResidueとして残る。使ってくれ」

### 3.2 選択肢の分岐強化

#### 慈悲ルートの追加
- `m_e04` に「話しかける」選択肢を追加。mercifulタグが5以上なら虚ろな衛兵が一瞬正気に戻り、地下墓地のヒントをくれる
- `f_e05` に「培養槽のN-06を弔う」選択肢を追加。empathyタグが蓄積され、コア預言者戦で特殊台詞が発生

#### 封印強化の意味付け
- `m_e07` の「封印を強化する」に `sets_flag: "seal_reinforced"` を追加
- 封印強化した場合、ボス戦で司教の攻撃力が低下する代わりに、真実の開示が遅れる（truth_stageが上がりにくくなる）
- **メカニクスとナラティブの一致**: 「真実を封じることで安全になるが、無知のまま」

#### 破壊ルートの追加
- `m_e08` の「書物を燃やす」に結果を追加。燃やすと次周回で灰の中から別の文書が見つかる（新イベント `m_e08b`）
- 「燃やしても残る」＝Residueのテーマを体現

### 3.3 伏線回収イベントの追加

#### 井戸の声の正体（中世）
- 新イベント `m_e05b`: well_peekedフラグ + loop >= 5 で発生。井戸の底に過去の周回の自分の死体がある。声は死体から発せられていた。
- 「お前は毎回ここを覗く。毎回驚く。そして毎回忘れる」

#### 礼拝堂の紋章の意味（中世）
- `m_e03` の紋章調査に `sets_flag: "emblem_examined"` を追加
- 新イベント `m_e03b`: emblem_examined + truth_stage >= 1 で発生。紋章はResidueプロジェクトのロゴだった。中世の礼拝堂に未来の紋章がある矛盾。
- クロスリンクの伏線として機能

#### 二つの声の回収（中世）
- 新イベント `m_e07b`: seal_broken + loop >= 5 で発生。封印を解いた後、二つの声の主が判明。一つは司教、もう一つは未来のコア預言者。時間を超えて同じ場所に縛られている。

### 3.4 NPCの人間化

#### 村長に名前と物語を
- 村長の名前: **アルダス**。かつての司教の弟子で、Residueの真実を知る最後の人間（だった）
- 実は彼自身が最初のResidue（N-01相当）。何十回もの周回で老い、記憶が混濁している
- 周回を重ねるごとに会話が変化し、最終的に「わしはお前の……いや、お前がわしの……」と混乱する

#### N-06を人間として描く
- N-06の名前: **カイ**。日記に名前を記載
- 日記の内容を段階的に開示し、カイの人間性を描く
  - 1段階目: 実験記録（客観的）
  - 2段階目: 感情の芽生え（「今日、空が見たいと思った」）
  - 3段階目: 反抗と諦め（「俺は人間じゃない。でも人間でありたかった」）

### 3.5 クロスリンクの深化

#### 新クロスリンク案: エリスの記憶
- 中世の村長の娘エリスの記憶が、未来の培養槽データに残っている
- 中世でエリスの話を聞く（m_e02c）→ 未来でエリスのデータを発見（f_e05b 新規）
- エリスがResidueプロジェクトの原点だったことが判明
- **感情的な繋がり**: 「一人の少女を救うために始まった実験が、無数の死を生んだ」

#### ジョブ持ち込み時の物語的影響
- 騎士で未来に行った場合: 鎧がN-01（アルダス）のものだったことが判明するイベント追加
- サイボーグで中世に行った場合: 機械部品が礼拝堂の紋章と共鳴し、プロジェクトの起源に触れるイベント追加

### 3.6 死に意味を持たせる感情設計

#### 死亡時の残留メッセージ
- 死亡した場所に次周回でメッセージが残る仕組みは設計されているが、イベントに反映されていない
- 各ノードに `death_residue` フィールドを追加。そのノードで死んだことがある場合に表示されるテキスト

#### 周回の重み
- 5周目以降、以前の周回で仲良くなったNPC（アルダス等）が「お前のことを覚えている」と言い始める
- ただし覚えているのはプレイヤーの行動パターンであり、記憶ではない。「お前はいつもここで立ち止まる。いつも同じことを聞く」

## 4. 優先度

| 優先度 | 改善項目 | 理由 |
|--------|---------|------|
| 🔴 高 | NPCの人間化（アルダス、カイ） | 感情フックが最も弱い部分。名前を付けるだけで大きく改善 |
| 🔴 高 | 井戸の声の回収（m_e05b） | 張りっぱなしの伏線で最も目立つ |
| 🔴 高 | 選択肢の意味付け（封印強化、慈悲ルート） | 選択が無意味だと周回のモチベーションが下がる |
| 🟡 中 | 村長の連鎖イベント（m_e02c, m_e02d） | ストーリーの連続性を大幅に改善 |
| 🟡 中 | N-06連鎖イベント（f_e03b, f_e03c） | 未来世界の感情軸になる |
| 🟡 中 | 紋章の意味回収（m_e03b） | クロスリンクの伏線として重要 |
| 🟢 低 | エリスのクロスリンク | 物語の深みは増すが、MVP後でも可 |
| 🟢 低 | 死亡残留メッセージ | システム側の実装が必要 |

## 5. その他の指摘

### 5.1 speaker設定の不整合
- `f_e06` の speaker が `"warrior"` になっているが、未来世界には warrior は存在しない。壁に刻まれたメッセージなので speaker は不要、または `"unknown"` にすべき
- `m_e05` の speaker が `"elder"` だが、井戸の声は「自分の声に似ている」と書かれており、`"unknown"` か `"self"` が適切
- `m_e_residue_reveal` の speaker が `"elder"` だが、これはボス（司教）の台詞のはず

### 5.2 テーマの統一性
- 中世世界のテーマ「信仰と封印」と未来世界のテーマ「実験と記録」は良い対比になっている
- しかし両方とも「真実に近づく」一方向しかない。**「真実を拒絶する」ルートが弱い**
- obedient/cautiousプレイに対する報酬が少なすぎる。「知らないほうが幸せだった」という結末も用意すべき

### 5.3 ローカライズの不足
- 新規イベント・NPC名のja.json追記が必要
- 現状のmemory_flagsに新フラグの追加が必要
